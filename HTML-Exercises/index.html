FEP1 HTML01 Assignments

Start de test op basis van cypress met het commando: npm run test:cypress

Installeer daarnaast AXE als extensie voor je browser en eventueel als Linter voor je IDE. Dit omdat de AXE gerelateerde foutmeldingen in cypress misschien te cryptisch zijn. De foutmeldingen zouden ook met de browser extensie te zien moeten zijn en geven je wellicht meer info omtrent de oorzaak van de fout.

Om de pagina's in de browser te bekijken moet je eerst de locale webserver starten met npm run dev en kun je vervolgens de assignment zonder cypress in de browser bekijken.
