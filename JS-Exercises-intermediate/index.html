FEP1 JS02-01 - JS Basics

Open de Terminal en start:

npm run test:vitest

Voor assignment02_1 en assignment02_2 vind je de files die je zou moeten aanpassen in de directory /src/utils.

Voor assignment02_3 dien je te beginnen met de klasse Trip in de file /src/model/Trip.js

Vervolgens dien je de methoden voor de klasse TripController (/src/controller/TripController.js) te geven.

Tenslotte ronden we deze assignment af met de render function in de file /src/view/showTrip. Merk op dat deze methode niet de TripController Klasse importeert, maar een instantie ervan welke in de file /src/controller/index.js wordt aangemaakt. We maken hier dus gebruik van een zogenaamde Singleton pattern om ervoor te zorgen dat als we meerdere bestanden in de view laag hebben die gebruik willen maken van de controller, ze allemaal gebruik maken van dezelfde instantie van de controller klasse.
